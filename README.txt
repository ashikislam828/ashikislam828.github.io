Website package for GitHub Pages.

Files included:
- index.html
- style.css
- research/index.html
- teaching/index.html
- publications/index.html
- programs/index.html
- seminars/index.html
- vitae/index.html

What to upload to your GitHub Pages repository:
1. Replace your current index.html with the included one.
2. Upload style.css.
3. Upload the folders: research, teaching, publications, programs, seminars, vitae.
4. Upload your CV as cv.pdf at the repository root.
5. Upload your headshot as profile.jpg at the repository root, then replace the placeholder on the homepage.

Optional:
- If your GitHub profile is not https://github.com/ashikislam828, update that link in the HTML files.
